# Log Analyzer

This Python tool analyzes log files and counts errors.

## Features

- reads log files
- counts total lines
- detects "error" entries
- counts errors
- handles invalid file paths

## Usage

Run:

python3 log_analyzer.py

Enter a file path, e.g.:
/var/log/syslog

## Technologies

- Python
- Linux

## Purpose

This project was created to practice Python, Linux and basic log analysis.
